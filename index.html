Engine Architecture

This system uses a dual-engine architecture to optimize recommendation accuracy:

  • Collaborative Filtering (40% Weight): Analyzes correlation vectors between users using Mean-Centered Cosine Similarity. Perfect for high-density shared tastes.
  • SVD Matrix Factorization (60% Weight): Projects user ratings into a 15-dimensional latent semantic space. Resolves matrix sparsity issues and extracts hidden structural associations.
